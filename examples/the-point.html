The Pipeline

Each one owns its scope. None overlap.

@blueprint
Design Architect
Interviews you. Synthesizes 12 design dimensions. Writes brand.css and the complete HTML structure with layout, color, typography, and every component decision.
@copy
Copywriter
Fills every [bracket placeholder] with audience-specific copy. Resolves LoremFlickr image URLs from the prose context. Never writes generic AI marketing language.
@review
QA Gate
Runs 20+ spec checks against the build. Files precise fix requests. Manages the @blueprint fix loop autonomously until zero failures remain.
@scout
Category Researcher
Surveys five competitors before @blueprint starts. Documents what every player does — so @blueprint can identify where to diverge and what the visual conventions actually are.
@forge
Component Builder
Creates new bp- components on demand when the layout needs something the design system doesn't have yet. Writes to components.css and registers in llm.md.
@artisan
Visual Refiner
Takes a screenshot of the built page. Identifies visual polish gaps that code review can't catch. Applies targeted refinements — spacing inconsistencies, weight mismatches, alignment errors.
