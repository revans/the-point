WHAT ORBIT SHOWS YOU

Every run, laid flat

Not a summary. Not sampled. The full sequence of what happened, in order, with numbers attached.

RUN TIMELINE

See the exact sequence of decisions your agent made

Open any run and walk through it step by step: which tool was called, what the agent was thinking before it called it, how long each step took, and where the plan changed. You stop debugging by re-running. You read what already happened.

Captures reasoning steps, not just tool calls — so you see the "why" before the "what." Works with Claude, GPT-4o, and any agent built on a structured framework.

Every step captured in sequence, not sampled
COST ACCOUNTING

Know exactly which runs — and which agents — are driving your bill

Orbit attributes cost to every run, every agent, every team. When your LLM spend goes up, you open Orbit and know immediately: which agent changed, which skill is now calling a more expensive model, which tool is generating unexpectedly long outputs. The bill stops being a mystery.

Cost is computed from actual token counts at the SDK level — not estimated from billing APIs. Accurate to the individual tool call.

Per tool call cost attribution, not just per run

Reasoning trace

See what the agent was thinking before it took each action. When it diverts from the plan you gave it, you read the reasoning step that caused it — not a log of what it did after.

Agent handoff tracing

Multi-agent runs surface as a single connected trace. You see what context Agent A passed to Agent B, where handoff latency was added, and whether the receiving agent got what it needed to continue the task.

Duration per run, per agent

After a skill update, compare run duration before and after. Orbit shows which tool call got slower and by how much — so you don't have to reproduce the slowdown to understand it.

HOW IT WORKS

One SDK. No agent code changes.

Orbit wraps your agent at the framework level. It observes what your agent does without touching what it does.

01

Install the SDK

One package. Works with Claude Code, LangChain, CrewAI, and custom agent frameworks. No changes to your agent logic.

npm install @orbitai/sdk
02

Wrap your agent

Two lines of initialization. Orbit instruments at the framework level — it sees every tool call, every LLM request, every skill invocation automatically from that point forward.

import { orbit } from '@orbitai/sdk'
orbit.init({ key: process.env.ORBIT_KEY })
03

Run your agent. See the trace.

Your first run appears in the Orbit dashboard before you've written a second line of integration code. Timeline, cost, tool calls — all there on the first run.

app.useorbit.dev/runs/first-trace
